# GoodBooks API

A FastAPI-based REST API for book ratings and recommendations, backed by MongoDB and using the GoodBooks-10k dataset.

## Features

- RESTful API with OpenAPI documentation
- MongoDB data storage with proper indexing
- Book search and filtering
- User ratings and to-read lists
- Tag-based book discovery
- Docker containerization
- Authentication with API keys
- Comprehensive logging
- Health checks and metrics
